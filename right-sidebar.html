    1. What HTML element is used to define a paragraph element?


    Answer:
    The "< p >" tag is the opening tag that signifies the beginning of a paragraph in an HTML document. The "< /p >" tag is the closing tag that indicates the end of a paragraph.

    When you wrap content between "< p >" and "< /p >" tags, you are essentially marking that content as a paragraph on the webpage. This allows web browsers to render the content as a distinct block of text, applying default styling (like margins) to separate it from surrounding content.

    Here's an example where we can apply the "< p >" and "< /p >" tag in our code:

    "< p >This website helps students to gain knowledge about code and programming < /p >"


    2. How can I create a clickable link in HTML that directs users to a specific website?


    Answer:
    To create a clickable link in HTML, you use the "< a >" (anchor) tag with the "href" attribute specifying the URL you want to link to. Here's an example:

    "< a href="https://www.youtube.com">Click this link "

    In this example, the text "Click this link" will appear as a clickable link, and when clicked, it will direct users to the website "https://www.youtube.com".


    3. How do you start and terminate an HTML document or code?

    Answer:

    We can start an HTML document, declaring the " < !DOCTYPE html > Following this declaration, you'll initiate the HTML document with the ' < html > ' tag. The code is divided into two sections: ' < head > ' and ' < body > '. The ' < head > ' section typically contains meta data, and the ' < body > ' section contains the actual content of your web page.


    Example :

    < !DOCTYPE html >
    < html >
    < head >

    < title > Page Title < /title >
    < /head >
    < body >

    < h1 > Web Design and Programming < /h1 >
    < /body >
    < /html >
